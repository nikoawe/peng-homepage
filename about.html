关于我

我是彭禹诺，贵阳市第三实验中学高中学生。这是一段从图形化编程积木块，到 C++ 算法竞赛一等奖的成长故事。

🔰 编程启蒙：图形化编程

最初接触编程，是从图形化编程（也称为图纸化编程或积木式编程）开始的。图形化编程将复杂的代码逻辑转化为可拖拽的彩色模块——就像搭积木一样组装程序。通过 Scratch 等图形化平台，我学会了程序的基本结构：顺序、分支、循环，以及事件驱动和消息传递。这段经历让我在正式接触代码之前，就已经建立起了计算思维的基础。图形化编程不是"玩具"，它是通往代码世界的桥梁——许多信息学竞赛选手的第一步都从这里迈出。

🐍 Python：从入门到认证

在图形化编程之后，我开始系统学习Python——一门简洁而强大的语言。Python 的语法干净明了，让我能快速将想法变成代码。我先后通过了中国电子学会（CIE）青少年软件编程等级考试 Python 1至4级。中国电子学会是工信部直属事业单位，其青少年编程等级考试是全国最具权威性的编程能力认证之一，由北京大学信息科学技术学院联合制定标准，腾讯提供技术支持。Python 共分 6 个级别：1-2 级侧重基础语法和 Turtle 绘图；3-4 级涉及函数、递归、分治算法与第三方库；5-6 级对接数据科学与人工智能。通过四级意味着我已掌握自定义函数、递归与分治算法思想、模块化编程以及第三方库的调用——这些正是人工智能和算法竞赛的重要基础。Python 也是我在全国青少年人工智能大赛省级选拔赛中获得一等奖（2026）的主力语言。

💻 C++：算法竞赛的武器

如果说 Python 是温柔的入门向导，那C++就是锋利的竞赛武器。为了冲刺信息学奥林匹克竞赛，我开始学习 C++ 并考取了中国电子学会青少年软件编程（C/C++）等级考试一级证书。虽然"随便考了个一级"——但 CIE 的 C/C++ 等级考试从一级到十级，一级本身涵盖了变量与赋值、基本输入输出（cin/cout）、算术表达式、逻辑运算、if/else 选择结构、for/while 循环结构、三目运算符等核心语法——能够独立编写完整的顺序、选择、循环结构程序。这不是"随便"，这是扎实的语法根基。正是这个基础支撑了我在 CSP-J/S 竞赛中取得一等奖。C++ 共十级，从一级的基础语法一路延伸到高级数据结构与算法，一级是坚实的第一步。

🏸 不只是代码

编程之外，我的生活同样丰富。打乒乓球、写书法、管理学生组织——每一件事都让我学会专注与坚持。2023 年贵阳市乒乓球联赛第一名，小学书法一等奖，以及学生自主管理委员会秘书长的工作经历——这些都是我的一部分。我相信全面发展的力量：在球台上练出的反应速度、在宣纸上养成的耐心、在学生工作中锻炼的领导力——最终都汇集到了我的代码中。
